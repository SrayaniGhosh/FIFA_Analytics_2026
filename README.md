📌 Project Workflow
🌐 Collected raw player and team statistics directly from the official FIFA website(https://www.fifa.com/en/tournaments/mens/worldcup/canadamexicousa2026) by manually copying and organizing the published tournament data.
🐍 Built a complete ETL pipeline in Python to clean, normalize, validate, and transform the raw text into a relational database.
🗄️ Designed and implemented a normalized MySQL database with multiple interconnected tables for teams, players, attacking, defending, distribution, discipline, goalkeeping.
🔍 Wrote SQL queries to uncover insights across team performance, player performance, tactical trends, defensive efficiency, passing networks, discipline, and goalkeeping.
📊 Created an interactive presentation to showcase the complete analysis.
🌍 Hosted the portfolio on GitHub Pages for easy access.

Key Highlights
⚽ Comprehensive analysis of 48 national teams and 1,242 players from the FIFA World Cup 2026.
📊 Team performance analysis across attacking, defending, ball distribution, discipline, and goalkeeping.
👤 Individual player performance analysis, including goals, assists, expected goals (xG), passing, defensive contributions, and disciplinary records.
📈 SQL-based rankings to identify the best-performing teams and players across multiple performance metrics.
🔍 Tactical insights using advanced football statistics such as xG, shot conversion, passing accuracy, possession, line-breaking passes, clean sheets, and goalkeeper actions.
🗄️ Normalized relational database designed to enable complex multi-table SQL analytics.
💡 Data-driven insights revealing attacking efficiency, defensive solidity, midfield control, player impact, and goalkeeping performance.
🌐 Web portfolio presenting all findings through visual dashboards, tables, and SQL-backed insights.
